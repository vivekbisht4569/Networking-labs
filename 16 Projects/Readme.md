## 🌐 Enterprise Networking Infrastructure Labs

A structured collection of hands-on networking labs focused on real-world enterprise network design and security concepts using Cisco Packet Tracer.

This repository includes practical implementations of:

- IP Addressing & Subnetting
- VLANs & VLAN Trunking
- Inter-VLAN Routing
- Router-On-A-Stick (ROAS)
- DHCP Configuration
- Spanning Tree Protocol (STP)
- Root Guard Security
- Access Control Lists (ACL)
- Enterprise Switching & Routing
- Network Troubleshooting

The labs are designed to simulate real enterprise environments such as banking and organizational infrastructures while improving practical networking and cybersecurity fundamentals.
