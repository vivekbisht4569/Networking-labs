# 🌐 Networking Labs

🚧 This repository is currently under development. Labs and write-ups are being added and improved regularly. 

---

## 🎯 Objective
- Build strong networking fundamentals  
- Gain practical configuration experience  
- Understand real-world network behavior  
- Prepare for cybersecurity and SOC roles  

---

## 📚 Topics Covered

- Basic Networking  
- IP Subnetting  
- VLAN & Trunking  
- Inter-VLAN Routing  
- Router-on-a-Stick  
- Routing (Static & Dynamic)  
- RIP & OSPF  
- NAT (Network Address Translation)  
- DHCP & DNS  
- ACL (Access Control Lists)  
- Port Security  
- STP (Spanning Tree Protocol)  
- Wireshark Analysis  
- Troubleshooting Scenarios  

---

## 🛠️ Tools & Technologies
- Cisco Packet Tracer  
- Wireshark  
- CLI Configuration (Router & Switch)
