# ARP Observation Lab

## Objective
To observe ARP process before ICMP communication.

## What Happens

1. PC0 sends ARP request (Broadcast).
2. PC1 replies with ARP reply (Unicast).
3. ICMP echo request is sent.
4. ICMP echo reply is received.

## Conclusion
ARP resolves MAC address before communication.
we also see the PDU details and OSI Model
