📘 IP Subnetting

🧠 What is IP Subnetting?

IP subnetting is the process of dividing a large network into smaller, manageable networks called subnets. It helps in better organization, security, and efficient use of IP addresses.

🎯 Why Subnetting is Used
Reduces network congestion
Improves performance
Enhances security by isolating networks
Efficient utilization of IP addresses

⚙️ Basic Concepts
IP Address → Unique identifier of a device
Subnet Mask → Defines network and host portion
CIDR (/24, /26, etc.) → Represents subnet size
Network Address → First address (not usable)
Broadcast Address → Last address (not usable)
Host Range → Usable IPs between network and broadcast
