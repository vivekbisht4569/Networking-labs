# 📘 Multi-Site Network Configuration

## 🛠️ Overview
Configured a multi-site network using Cisco Packet Tracer by subnetting a /24 network and enabling communication between different locations.

---

## 🎯 What I Did
- Subnetted **192.168.1.0/24 into 4 subnets**
- Connected two different sites (**Site 1 & Site 2**)
- Configured routers for inter-site communication (WAN links)
- Implemented DHCP for automatic IP assignment
- Connected internal network to an external (internet) router
- Verified connectivity across all devices

---

## 🌐 Network Design
- Two local sites with switches and PCs
- Routers connected via serial links
- Central router acting as internet gateway
- Servers used for testing connectivity

---

## ⚙️ Key Configuration
- IP addressing using subnetting
- Router interface configuration
- Serial link setup between routers
- DHCP configuration
- Routing between multiple networks

---

## 📚 What I Learned
- Multi-site network design
- Subnetting in real-world scenarios
- Router-to-router communication (WAN)
- DHCP working in networks
- End-to-end connectivity testing

---

## 🚀 Result
- Successfully established communication between both sites
- Verified network connectivity across all devices
