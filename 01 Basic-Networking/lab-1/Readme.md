Basic LAN Communication

 Objective:

To establish communication between two devices within the same network using a switch.

What I Did:
Connected 2 PCs using a switch
Assigned IP addresses manually
Verified connectivity using ping

 Configuration
PC0 → 192.168.1.10 /24
PC1 → 192.168.1.20 /24

 Result:
Successful communication between both PCs


What I Learned:
Devices in the same subnet can communicate without a router
Role of a switch in forwarding frames
Importance of correct IP addressing
